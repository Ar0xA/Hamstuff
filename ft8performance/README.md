Comparing various ft8 decoders for funsies.

compare_decodes.py
------------------
Program to compare wsjtx_all.txt, jtdx_all.txt, mshv_all.txt and nexus_all.txt and print stats about decode quality and quantity.

play_ft8.py
-----------
Program to play a 48khz sampled .wav to a virtual audio cable input to be ingested by ft8 decoder programs. You play with offset to get the DT within tolerances you want.

compare_wsjtx.py
----------------
Program to compare decoding quality of various WSJTX settings. Use the same recording for valid comparison. Ingests wsjtx_norm.txt, wsjtx_2pass.txt, wsjtx_3pass.txt and wsjtx_.late.txt and prints some stats about the decoder performaces.
